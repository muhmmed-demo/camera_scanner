# Camera Scanner - مكتشف الكاميرات 📷

تطبيق أندرويد لاكتشاف كاميرات المراقبة (IP Cameras) المتصلة بالشبكة المحلية.

## المميزات
- اكتشاف كاميرات ONVIF تلقائياً عبر بروتوكول WS-Discovery
- فحص المنافذ الشائعة: RTSP (554), HTTP (80), ONVIF (8000), وغيرها
- واجهة مستخدم عصرية بدعم كامل للغة العربية
- متوافق مع أندرويد 8 وما فوق

## متطلبات التشغيل
- هاتف أندرويد 8.0 (Oreo) أو أعلى
- الاتصال بنفس الشبكة المحلية (Wi-Fi) التي تتصل بها الكاميرات

## بناء التطبيق
يتم بناء التطبيق تلقائياً عبر GitHub Actions. كل ما تحتاجه:
1. رفع هذا المجلد كـ Repository على GitHub
2. انتظر 15-20 دقيقة
3. اذهب إلى تبويب Actions → اختر آخر عملية بناء → حمّل الـ APK من قسم Artifacts

## تنبيه قانوني
هذا التطبيق مخصص للاستخدام على شبكتك الخاصة فقط، أو على أي شبكة لديك تصريح بفحصها.
